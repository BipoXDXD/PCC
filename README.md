# Python Crash Course (PCC)

A repository for Python Crash Course exercises and projects.

## Structure

This repository contains 20 chapter directories (`ch01` through `ch20`), each with exercises and code from the corresponding chapter of Python Crash Course.

## Setup

1. Clone the repository:
```bash
git clone https://github.com/BipoXDXD/PCC.git
cd PCC
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On macOS/Linux
# or
venv\Scripts\activate  # On Windows
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running Tests

This project uses pytest for testing. To run the tests:

```bash
pytest
```

Or for verbose output:

```bash
pytest -v
```
